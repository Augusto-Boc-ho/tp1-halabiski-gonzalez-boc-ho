# tp1-halabiski-gonzalez-boc-ho
## Install

```bash
python3 -m venv venv
source venv/bin/activate
pip install flask
pip install flask-cors
pip install Flask-SQLAlchemy
pip install psycopg2-binary
pip install flask-login
```

## Run

```bash
source venv/bin/activate
cd backend
flask --app main.py run --debug
```
